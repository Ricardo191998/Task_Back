# Correr el proyecto

Instalar dependencias de node 
 npm install

Creear una carpeta env y dentro de ella un archivo .env con los valores de las variables de entorno

Ejemplo :

SALT_WORK_FACTOR = 10
PORT = 4000


Correr 

npm run start 


Ojo 

Requiere de una base de datos de mongo corriendo en local
